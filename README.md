# 📱 PaperPilot - Navigate Your Past Papers

---

## About PaperPilot

PaperPilot is a mobile application designed to help university students access past examination papers easily and efficiently. Instead of searching through WhatsApp groups, emails, and university portals, students can find all past papers in one organized place.

The app provides a simple, intuitive interface that allows students to browse papers by their program and semester. With features like offline access and automatic updates, PaperPilot ensures that students always have the resources they need to prepare for their exams.

---

## Key Features

### 📚 Program & Semester Navigation
- Browse past papers organized by program (e.g., BS(IT), BS(CS))
- View papers by semester (1-8)
- Pre-selected program and semester based on user profile

### ⭐ Offline Favorites
- Save papers to favorites
- Download PDFs for offline study
- Access saved papers anytime, anywhere without internet

### 🔄 Automatic Updates
- New papers appear automatically
- No need to manually check for updates
- Stay informed about new study materials

### 🔍 Smart Search
- Real-time search on all screens
- Find programs, semesters, and papers instantly
- Case-insensitive search for convenience

### 📄 Integrated PDF Viewer
- Read papers directly within the app
- Zoom in/out and navigate pages smoothly
- Stream from Google Drive or load offline

### 👤 User Profile
- One-time signup on first launch
- View and edit profile anytime
- Personalized experience based on your program and semester

### ⭐ Dedicated Favorites Screen
- All favorite papers in one place
- Quick access to saved papers
- Search within favorites
